# Innovision 2.0

A modern, interactive event website built with React 19, Vite, and advanced animation libraries. This project features a premium UI with smooth scrolling, custom cursors, and 3D elements.

- **Modern Tech Stack**: Built with React 19 and Vite for lightning-fast performance.

- **Advanced Animations**: Powered by **GSAP** and **Framer Motion** for complex timeline-based and layout animations.
- **3D Elements**: Utilizes **Three.js** via `@react-three/fiber` and `@react-three/drei` for immersive 3D content.
- **Custom UI**: Features a custom cursor, preloader, and a responsive design using **TailwindCSS**.
- **Interactive Components**:
  - Hero Section
  - Events Showcase
  - Custom Navbar & Footer
